Technical Notes for this page style

  • Basic styles implementation.
  • Page design.
  • Multi-column and media-query css with css3.
  • prefix-free for cross-browser css3 support.
  • On WebKit-based/Chrome a custom scroller is designed.
  • Syntax highlighting(and code editor) implemented with ACE.
  • MathJax $\LaTeX$ support.
